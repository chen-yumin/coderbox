CoderBox.io

CoderBox.io is a progressive web app designed by developers for developers. It contains all the common tools needed for day to day coding development.

CoderBox Modules

Epoch Time Converter

Epoch Time Converter can help you easily convert from Unix (Epoch) time to human readable time. Or from a human readable timestamp to the miliseconds since Epoch time.

Floating Point Converter

Floating Point Converter can help you easily convert single (32-bit) or double (64-bit) precision floating-point values to its IEEE Standard 754 floating point in binary and hexdecimal representation.
